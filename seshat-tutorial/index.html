Tutorial

This tutorial demonstrates the creation of a knowledge graph of world history with TerminusDB. It is based on an RDF dataset that was produced by the Seshat project in 2018, as part of the European Commission's Horizon 2020 Aligned project. Seshat researchers are continuously improving, updating and expanding this data - the latest releases can be found on the Seshat data site

Requirements

To run this tutorial you will need to install your own copy of TerminusDB. The two easiest options are either:
  1. Download and install your own TerminusDB with docker
  2. Deploy online to Heroku
